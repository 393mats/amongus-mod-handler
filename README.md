# AmongUs Mod Handler

PC版AmongUsのModのインストールやModの切り替えをGUI操作で行えるツール。
今後種類や追加が行えるように改修する予定。

### 対応Mod

- [The Other Roles]](https://github.com/Eisbison/TheOtherRoles)
- [The Other Roles GM Edition]](https://github.com/yukinogatari/TheOtherRoles-GM)
